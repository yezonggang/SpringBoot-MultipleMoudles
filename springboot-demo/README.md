Ctrl+Y  删除当前行  
Ctrl+d  复制当前行  
Ctrl+Alt+Enter  格式补全后，换行  
Ctrl+Enter  光标到下一行  
Ctrl+p  参数展示  
Home 光标到行首  
End 光标到行尾  
Ctrl+Shift+n 全局搜索文件  
Ctrl+n 全局搜类  
Ctrl+Shift+R 全局搜索替换  
Ctrl+Shift+F 全局搜索  
Ctrl + Shift + l   注释选中代码块  
Ctrl + z   回滚
